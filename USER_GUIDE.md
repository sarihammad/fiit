## Meal Logging

### Photo Logging

**Step-by-Step Process:**

1. **Open Camera**
   - Tap the camera button on the home screen
   - Grant camera permissions if prompted
   - Position your meal in the frame

2. **Take Photo**
   - Ensure good lighting
   - Keep the camera steady
   - Include the entire meal
   - Tap the capture button

3. **AI Analysis**
   - Wait for AI to process the image
   - Review the food identification
   - Check the confidence level
   - Verify the nutrition information

4. **Confirm Results**
   - Select the correct food if multiple options appear
   - Adjust portion size if needed
   - Add any missing items
   - Save your meal

**Confidence Levels:**

- **High (80%+)**: AI is very confident - usually auto-accepted
- **Medium (50-80%)**: AI is confident - review and confirm
- **Low (<50%)**: AI is uncertain - manual selection required

### Manual Entry

**When to Use Manual Entry:**

- AI couldn't identify the food
- You want to add specific details
- You're logging a recipe
- You need to adjust nutrition values

**Manual Entry Process:**

1. Tap "Add Manually" on the meal logging screen
2. Search for your food item
3. Select the correct match
4. Adjust portion size
5. Add any additional items
6. Save your meal

### Portion Size Adjustment

**Portion Options:**

- **Small**: 0.5x standard serving
- **Medium**: 1x standard serving (default)
- **Large**: 1.5x standard serving
- **Custom**: Enter specific weight or volume

**Portion Tips:**

- Use common objects for reference (deck of cards = 3oz meat)
- Weigh food when possible for accuracy
- Consider cooking method (raw vs. cooked)
- Account for sauces and seasonings

**Lighting:**

- Use natural light when possible

- Use flash only if necessary

**Composition:**

- Fill the frame with your meal
- Avoid cluttered backgrounds
- Keep the camera steady
- Take multiple angles if needed

**Timing:**

- Take photos before eating
- Capture the meal as served
- Include all components
- Don't forget beverages

### Accuracy Tips

**Portion Estimation:**

- Use your hand as a reference

- Weigh food when possible

## Troubleshooting

### Common Issues

**Camera Not Working:**

1. Check camera permissions in settings
2. Restart the app
3. Restart your device
4. Update the app to latest version

**AI Not Recognizing Food:**

1. Ensure good lighting
2. Take a clear, focused photo
3. Include the entire meal in frame
4. Try manual entry as backup

**App Crashes:**

1. Close and reopen the app
2. Restart your device
3. Check for app updates
4. Contact support if issue persists

**Sync Issues:**

1. Check internet connection
2. Force close and reopen app
3. Log out and back in
4. Contact support if data is missing

### Performance Issues

**Slow Loading:**

1. Check internet connection
2. Close other apps
3. Restart the app
4. Update to latest version

**Battery Drain:**

1. Close app when not in use
2. Disable unnecessary notifications
3. Reduce background app refresh
4. Update to latest version

**Storage Issues:**

1. Clear app cache
2. Delete old photos
3. Export and delete old data
4. Update to latest version

### Data Issues

**Missing Meals:**

1. Check if meals are in offline queue
2. Ensure internet connection
3. Try logging the meal again
4. Contact support if data is lost

**Incorrect Nutrition:**

1. Verify food selection
2. Check portion size
3. Use manual entry for accuracy
4. Report incorrect data to support

**Sync Problems:**

1. Check internet connection
