# FIIT Operator Checklist

This checklist validates the execution‑coach loop end‑to‑end.

## Core Flow

1) **Start**
   - Enter a vague goal.
   - Verify the first clarification question appears.

2) **Clarify**
   - Answer 3–7 questions.
   - Verify progression to Plan after max questions.

3) **Commit Plan**
   - Generate preview.
   - Commit plan and confirm it locks.
   - Verify navigation to Today Mode.

4) **Today Mode**
   - Confirm only 1–3 tasks appear.
   - Tap **Start** to mark a task done.
   - Defer a task 3 times and confirm **Make this easier** appears.

## Limits & Monetization

1) **Micro‑step limit**
   - Use micro‑step rewrites until limit is reached.
   - Verify upgrade CTA appears.

2) **Plan reset limit**
   - Reset plan until limit is reached.
   - Verify upgrade CTA appears on the plan screen.

## Notifications

1) Enable notifications in Settings.
2) Confirm daily execution reminder is scheduled.
3) Send a test notification.

## Regression Checks

- App opens to **Today** when a locked plan exists.
- App opens to **Plan** when a goal exists but no plan.
- App opens to **Start** with no goal.
